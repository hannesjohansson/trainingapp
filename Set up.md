Setting up Node.js / Express
===========

- Download node.js
- Create folder APP and open it in the terminal
- npm install -g express
- express --sessions --ejs web
- npm install
(last step creates node modules)

Starting mongoDB
===========
- cd /usr/local/mongodb
- ./bin/mongod
